# シフト作成ツール

ちょっと習った貧欲法でつくってみた

Excelを読み込み、Flaskの画面から業務設定を入力して、1日分のシフトを自動生成

- 入力Excel: `namesheet.xlsx`
- 列: `名前`, `学年`, `委員会` + `9-10` のように `-` を含む時間列
- 可否: `○` / `×`


## 出力
- 画面表示: テーブルで表示  
- Excel出力: `shift_output.xlsx` をダウンロード


- 出席不可の時間には配属しない  
- 同じ時間に同じ人を複数業務へ配属しない  
- 各「時間×業務」に4年生を最低1人（確保できない場合は警告）  
- 3連続勤務を禁止（最大2連勤）  
- 1〜3年生はなるべく学年バランスが偏らないように割当
